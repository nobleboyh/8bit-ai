The RAGAS metrics measure different aspects of LLM response quality: Faithfulness checks whether the response is factually supported by the provided context. Context Precision measures whether relevant context chunks appear higher in the ranking. Context Recall measures whether all relevant context chunks were retrieved. Each metric is scored from 0 to 1. KG-aided responses include source code + knowledge-graph structural metadata, while source-only responses include source files without KG metadata.
